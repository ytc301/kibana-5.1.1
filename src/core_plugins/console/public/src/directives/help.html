帮助
你可以在编辑窗口中键入一个或多个请求，控制台会理解简介的格式请求，通过点击绿色三角按钮可以单独运行其中一个：
基本编辑功能
Ctrl/Cmd + I
自动缩进当前的请求
Ctrl + Space
打开自动补全（即使没有编辑）
Ctrl/Cmd + Enter
提交请求
Ctrl/Cmd + Up/Down
跳转到上一个或下一个请求的开始或结束
Ctrl/Cmd + Alt + L
折叠或打开当前代码块
Ctrl/Cmd + Option + 0
折叠除当前代码块以外的所有代码块，通过增加一个移位来扩展
当打开自动补全功能时
Down arrow
把光标切换到自动补全菜单，进一步选择项
Enter/Tab
在自动补全菜单中选择当前项或第一项补全
Esc
关闭自动补全菜单
